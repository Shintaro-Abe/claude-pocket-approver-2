# 👻 Boo Device — Claude Code 許可デバイス

M5StickC PLUS2 で動作する Claude Code 承認リクエストデバイスです。

ブー(Boo)はただの承認ボタンではありません。**たまごっち**です。
承認するとご飯をあげられ、放置すると空腹になり、Claudeが作業するほど疲れていきます。
ブーを育てながら Claude Code を使いましょう。

```
承認する          →  fed ↑↑     ブーにご飯をあげる 🍖
否認する          →  fed ↓      ブーがさびしがる 😢
タイムアウト放置  →  fed ↓↓     ブーが空腹で泣く 😱
作業中            →  energy ↓   Claudeが頑張るほど疲れる ⚡
スリープ中        →  energy ↑   ブーがお昼寝で回復 💤
なでる (アイドル中Bボタン)  →  fed ↑   ブーがときめく 💕
```

```
└── README.md        # このファイル
```

## 1. Arduino セットアップ

### 必要ライブラリ (Arduino IDE)
- **M5StickCPlus2** (M5Stack 公式)
- **ArduinoJson** (v6.x)

### ボード設定
- ボード: `M5StickC`  
- Upload Speed: `1500000`

### 書き込み
1. `boo_device.ino` を Arduino IDE で開く
2. M5StickC PLUS2 を USB 接続
3. ポートを選択してアップロード

---

## 2. Python ブリッジセットアップ

```bash
pip install pyserial mcp
python boo_bridge.py
# ポートを自動検出します
# 手動指定の場合:
python boo_bridge.py --port /dev/cu.usbserial-XXXX
```

---

## 3. Claude Code への登録

`~/.claude/settings.json` に追記:

```json
{
  "mcpServers": {
    "boo-approval": {
      "command": "python",
      "args": ["/path/to/boo_bridge.py"],
      "env": {}
    }
  }
}
```

または `claude mcp add` コマンドで登録:

```bash
claude mcp add boo-approval python /path/to/boo_bridge.py
```

---

## 4. Claude Code での使い方

`.claude/CLAUDE.md` に以下を追加すると、Claude が自動的に承認を求めるようになります:

```markdown
## 重要なルール
- ファイルの削除・上書きの前に `approve_request` ツールで承認を求めること
- 外部コマンド実行前に `approve_request` を使うこと
- danger: true は rm, curl, git push など不可逆操作に使うこと
- 作業開始時に `notify_working`、終了時に `notify_idle` を呼ぶこと
```

## 5. ボタン操作

| 状態 | ボタンA | ボタンB |
|------|---------|---------|
| 承認リクエスト中 | ✅ 承認 (fed +2 🍖) | ❌ 否認 (fed −1 😢) |
| アイドル | 📊 統計画面へ | 💕 なでる (fed +0.5) |
| スリープ | ▶ 起動 (napped 時間を記録) | — |
| 統計画面 | ⬅ 戻る | 🔄 統計リセット (fed/energy は保持) |

> **Note:** タイムアウト放置は fed −2 のペナルティがあります。承認リクエストは放置しないでください。

## 7. パラメータ仕様

### fed（ご飯ゲージ）— ハート 0〜8個

| イベント | 変化量 | 備考 |
|----------|--------|------|
| 承認 (A ボタン) | **+2.0** | ご飯をあげる 🍖 |
| なでる (アイドル中 B ボタン) | **+0.5** | スキンシップ 💕 |
| 否認 (B ボタン) | **−1.0** | さびしがる |
| タイムアウト放置 | **−2.0** | 空腹で泣く |
| 時間経過 | **−0.15/分** | 自然減衰 |

fed が **1未満** で `ART_HUNGRY`（空腹顔）に切り替わります。

### energy（エネルギーゲージ）— ブロック 0〜6個

| 状態 | 変化量 | 備考 |
|------|--------|------|
| スリープ中 | **+0.8/分** | お昼寝で回復 💤 |
| アイドル中 | −0.05/分 | 微減 |
| 作業中 | **−0.4/分** | Claudeが頑張るほど消費 ⚡ |
| トークン 1K ごと | **−0.02** | 重い作業ほど消費 |

energy が **0.5未満** で `ART_TIRED`（疲弊顔）に切り替わります。

### mood（機嫌ゲージ）— ハート 0〜8個

```
mood = (fed/8 × 60%) + (energy/6 × 40%)
```

| mood | アイドルアニメ |
|------|---------------|
| 0〜2 | 疲れ顔・地味 |
| 3〜5 | 通常 + ときどきときめき |
| 6〜8 | ときめき・大好き・ドキドキ全開 💘 |
